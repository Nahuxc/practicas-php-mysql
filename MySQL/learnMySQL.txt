* ¿Que es una base de datos? *
- una base de datos es un conjunto de datos pertenecientes a un mismo contexto y almacenados sistematicamente para su posterior uso.


* elementos que componen una base de datos *

- Tablas --> representa una entidad.
- Columnas --> representa los campos.
- Filas --> representa una ocurrencia de la entidad el cual guarda sus datos del objeto de cada campo.

ejemplo:

DATABASE --> BLOG:

- TABLA --> USUARIO --> //entidad usuario

- cada columna representa cada atributo que va a tener esa entidad(Usuario).

- cada fila representa un objeto de esa entidad(Usuario).
